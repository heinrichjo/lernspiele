2. Grundsätze

3. Datenverarbeitung durch externe Dienste

Beim Besuch unserer Website werden aufgrund der technischen Einbindung externer Inhalte oder Links Daten an Dritte übermittelt:

a) Tailwind CSS (CDN-Nutzung)

Zur optimalen Darstellung der Website-Elemente und Stylesheets wird die Bibliothek Tailwind CSS über ein Content Delivery Network (CDN) geladen.

  • Verarbeitete Daten: Hierbei wird Ihre IP-Adresse an den Betreiber des CDN übermittelt, damit die Stylesheets an Ihren Browser gesendet werden können.
  • Zweck: Bereitstellung der notwendigen Design-Elemente für eine funktionale Website.
  • Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der technischen Funktionsfähigkeit und der Optimierung der Darstellung).

b) Externe Bildquelle (Logo)

Das Logo wird von der externen Website der Welserschule (www.welserschule.de) geladen.

  • Verarbeitete Daten: Ihre IP-Adresse wird an den Server der Welserschule übermittelt, um das Bild zu laden.

c) Verlinkte Lernspiele (Hosting über GitHub Pages)

Alle "Spiel starten"-Links auf dieser Website führen zu den auf **GitHub Pages** gehosteten Lernspielen (z. B. auf der Domain https://heinrichjo.github.io/lernspiele/...).

  • Verarbeitete Daten beim Klick: Beim Klick auf den Link verlassen Sie unseren Verantwortungsbereich. Die Übermittlung von Daten (insbesondere Ihrer IP-Adresse) an die Server von GitHub (ein Dienst von Microsoft) erfolgt erst, wenn Sie das jeweilige Spiel aufrufen.
  • Zweck: Bereitstellung der Spiele-Prototypen auf einer stabilen Hosting-Plattform.
  • Hinweis: Sobald Sie die externe Seite aufgerufen haben, gelten die Datenschutzbestimmungen von GitHub.

d) Besucher- und Klickzähler (Firebase Firestore)

Die Website verwendet Firebase Firestore (ein Dienst von Google) zur internen, pseudonymen Erfolgsmessung und Nutzungsanalyse, um die didaktische Relevanz der Inhalte zu prüfen.

  • Verarbeitete Daten: Es werden lediglich aggregierte Zählungen (Anzahl der Seitenaufrufe, Anzahl der Klicks auf einzelne Spiele) gespeichert. Es erfolgt **keine Speicherung von IP-Adressen, Nutzer-IDs oder sonstigen personenbezogenen Daten** in Firestore.
  • Cookies/Speicher: Es wird der **Session Storage** Ihres Browsers verwendet, um zu verhindern, dass die Seitenaufrufe innerhalb einer Sitzung mehrfach gezählt werden. Dieser temporäre Speicher wird beim Schließen des Browsers gelöscht.
  • Zweck: Statistische Auswertung zur Optimierung des Lehrangebots.
  • Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der Optimierung der Dienste, da die Verarbeitung pseudonymisiert erfolgt).
